## About

**Portafolio** is a multi-agent market analysis system by **Cristhian White | Diggital Accelerator**.

It spawns 5 specialized research agents (Crypto, Stocks, Forex, Commodities + Chief Strategist), adapts to your personal risk profile and watchlist, and generates an interactive branded dashboard with 3 visual themes.

## Quick Start

### 1. Install

```bash
curl -sL https://raw.githubusercontent.com/YOUR_GITHUB_USERNAME/portafolio/main/install.sh | bash
```

### 2. Configure

```
/portafolio-setup
```

### 3. Run your first analysis

```
/portafolio
```

## Features

- **5 parallel AI agents** — Crypto, Stocks, Forex, Commodities + Chief Strategist
- **Personal watchlist** — track your own assets alongside the general analysis
- **Local currency** — see prices in your currency (COP, MXN, EUR, etc.)
- **3 dashboard themes** — Light, Dark Fintech, Dark Terminal
- **Risk-adjusted recommendations** — tuned to your Conservative/Moderate/Aggressive profile
- **Historical accuracy tracking** — compare past recommendations to real outcomes
- **Bilingual** — English and Spanish
